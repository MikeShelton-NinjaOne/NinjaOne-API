# NinjaOne-API
Heads up: These scripts aren't officially supported by NinjaOne. They're shared here to help out the community, but if something breaks, NinjaOne Support won't be able to help you sort it out. Test things in a safe environment first, use at your own risk, and feel free to open a PR if you've got improvements to share.

Commonly requested use cases will be added to the repo over time, so check back now and then to see what's new.
